    Absolute lengths

  • Centi meter (0.5cm)
  • Millimeter (5mm)
  • inches (0.2in)
  • Pixel (10px)
  • points(10pt)
  • picas(pc)

    Relative lengths

  • em(emphosize)-relative to the font size of the element(border:2em solid)
  • ex-relative to the current x height of the current font
  • ch- relative to the width of 0 of the choosen font size(border:2ch solid; 2*width of the 0 in the choosen font size)
  • rem- root em. relative to the font size of the root element(font-size: 1rem;)
  • vw- related to the viewport width(font-size: 3vw;)
  • vh- related to the viewport height(font-size: 3vh;)
  • vmin
  • vmax
  • %- relative to the parent element(margin: 6%;)
