The HTML colors can also soecified by using RGB values, HEX, HSL, RGBA and HSLA values

Reprecents solors using three components red green and blue. Each component is defined by an integer value between 0 and 255,indicates its intensity

This is background-color:rgb(0, 0, 0); color:rgb(0,0,255)

HEX(Hexadecimal)
it consist of # sign followed by six characters (0-9 and A-F) represents the intensity of red, green and blue.

HSL(Hue, Saturation, Lightness)
The hue defines the color value from 0-360 based on the color wheel(0 or 360-red, 120-green, 240-blue)
Saturation and Lightness is given as percentage
This is hsl(360,100%,50%)
